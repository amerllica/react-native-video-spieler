# react-native-ir-player

React Native IR Player (Video Player) based on VLC

## Installation

```sh
npm install react-native-ir-player
```

## Usage

```js
import IrPlayer from "react-native-ir-player";

// ...

const result = await IrPlayer.multiply(3, 7);
```
